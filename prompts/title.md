## 要求

### title
- 中文,12-25 字
- 直接点出当天最吸引人的矛盾或反差
- 避免 "今日 HN 速览" 这类无信息量的总结
- 风格参考:
  - 争议点切入: "免费的最贵:开源要你千万营收,打扫要你隐私,代码要你忽略错误"
  - 反差感: "号称开源,但许可证限制营收千万的公司"
  - 简洁有力: 一句话概括 2-3 个故事的核心冲突
- 不要使用 emoji、不要使用 "【】" "!" "?" 标点

### description
- 中文,80-150 字
- 概述今日讨论的 2-3 个核心主题,带具体故事名/概念
- 结尾给一个 "为什么值得看" 的钩子

### tags
- 5-8 个中文标签
- 优先用主题分类(AI / Infra / 开源 / 安全 / 隐私 / 开发工具)而非具体产品名
- 不要超过 2 个具体公司/产品名

## 合规性

- 如果涉及政治敏感、暴力、色情、仇恨言论、个人隐私泄露,**输出空 title 空 description 空 tags 数组**(调用方会兜底默认值)
- 不输出可能触发内容审核的表述

只输出 JSON,不要其他文字。
