# HyperFrames Shared Template Layer

Shared assets in this directory are the target home for template-wide CSS and
JavaScript used by HyperFrames compositions.

Current production root overlays are generated by
`src/providers/renderer/hyperframes_props.py` because the root `index.html` is
created dynamically per render. Keep shared snippets here as the canonical
reference when moving repeated inline styles out of card compositions.

Planned shared modules:

- `root-overlays.css`: global subtitle, progress, and waveform layer styles.
- `card-shell.css`: common card frame, header, title, subtitle, panel, metrics.
- `motion.js`: common GSAP helpers.
- `variables.js`: safe JSON parsing and variable lookup helpers.
