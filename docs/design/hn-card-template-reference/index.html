HN TechPulse 卡片模板

四张 1280x720 卡片共用同一套设计系统：统一的暖白背景、HN 橙色品牌线、页眉、指标胶囊、标签和底部安全区。每张卡仍是独立的 Mustache 模板，可以直接打开预览，也可以由 Python 注入数据渲染。

1280x720 16:9 1 design system 4 templates Mustache-compatible
1 openingcard.html opening / 每日 3 条头条
2 eventcard.html event / 单个故事详情
3 atmospherecard.html atmosphere / 争议与评论氛围
4 closingcard.html closing / 今日回顾
直接打开 index.html 或任一卡片 HTML 即可预览，无需启动服务器。卡片内置 sample-data，浏览器会通过 design/render.js 自动渲染模板变量。
